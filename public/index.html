Powerful Image Analysis Features

Comprehensive computer vision capabilities powered by Microsoft Azure's state-of-the-art AI models.

Smart Descriptions

Generate human-readable descriptions of images with confidence scores. Perfect for accessibility and content management.

Intelligent Tagging

Extract relevant tags and keywords automatically. Ideal for search optimization and content categorization.

Object Detection

Detect and locate objects with precise bounding boxes and confidence levels for detailed analysis.

Color Analysis

Analyze dominant colors, accent colors, and color schemes for design and branding applications.

Face Detection

Detect faces with age and gender estimation capabilities for demographic analysis and personalization.

Brand Recognition

Identify well-known brands and logos in images for marketing analysis and brand monitoring.

# Analyze image from URL
curl -X POST http://YOUR-API-URL/api/v1/analyze/url \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com/image.jpg",
    "features": ["Description", "Tags", "Objects"]
  }'

# Upload and analyze image file
curl -X POST http://YOUR-API-URL/api/v1/analyze/upload \
  -F "image=@/path/to/image.jpg" \
  -F 'features=["Description","Tags"]'

python 
import requests

# Analyze image from URL
url = "http://YOUR-API-URL/api/v1/analyze/url"
data = {
    "url": "https://example.com/image.jpg",
    "features": ["Description", "Tags", "Objects"]
}
response = requests.post(url, json=data)
result = response.json()

# Upload and analyze image file
files = {'image': open('image.jpg', 'rb')}
data = {'features': '["Description","Tags"]'}
upload_response = requests.post(
    "http://YOUR-API-URL/api/v1/analyze/upload",
    files=files,
    data=data
)
upload_result = upload_response.json()
powershell 
# Analyze image from URL
$body = @{
    url = "https://example.com/image.jpg"
    features = @("Description", "Tags", "Objects")
} | ConvertTo-Json

$response = Invoke-RestMethod -Uri "http://YOUR-API-URL/api/v1/analyze/url" `
    -Method Post -Body $body -ContentType "application/json"

# Display results
$response | ConvertTo-Json -Depth 10
